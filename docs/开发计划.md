## 六、风险提示

1. **AI 模块接入**需要选择合适的 LLM 服务提供商，涉及 API Key 申请和费用预算
2. **拍卖模块**的并发出价控制是技术难点，需要仔细设计分布式锁方案
3. **文件上传**（头像/合同PDF）需要决定存储方案（本地 vs OSS）
4. **支付对接**目前保证金充值/提现均为 Mock，真实对接需要申请支付渠道
5. `application.yml` 中 JWT Secret 和数据库密码硬编码，上线前必须改为环境变量
